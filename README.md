# RichText

UGUI RichText

## 实现方式

继承至原Text，扩展Text原生支持的<material>进行标签检测。

## 主要功能

* 图文混排
* 下划线
* 阴影
* 描边
* 颜色渐变
* 其中混排图片和下划线文字支持点击事件响应
* 支持各种标签(包括Text原生标签如<size>等)混合使用
